## Project Overview

AceReStreamer scrapes AceStream links from various sources (HTML pages, IPTV m3u8 files, APIs) and re-streams them via HLS. It supports IPTV clients (Xtream Codes / m3u8), desktop players (VLC/IINA/MPV), and an in-browser player.

## Commands

### Backend (Python 3.14+, uv)

```bash
uv run pytest                        # Run all tests
uv run pytest tests/api/test_login.py  # Run a single test file
uv run pytest -k "test_name"         # Run tests matching a pattern
uv run mypy                          # Type check (strict)
uv run ty check .                    # Additional type checker
uv run ruff check --fix              # Lint and auto-fix
uv run ruff format                   # Format code
uv sync --extra type --extra test    # Install dev dependencies
```

### Frontend (React 19, TypeScript, bun)

```bash
cd frontend
bun install                          # Install dependencies
bun run dev                          # Vite dev server
bun run build                        # Production build
bun run build-aio                    # Build into acere/dist/ (all-in-one mode)
bun run typecheck                    # tsc --noEmit
bun run lint                         # Biome check --write --unsafe
```

### Client Code Generation

```bash
bash scripts/generate-client.sh      # Regenerate frontend/src/client/ from OpenAPI spec
```

Requires an active virtualenv. Sets `IN_OPEN_API_MODE=true` to extract the OpenAPI spec from the FastAPI app without side effects, then runs `@hey-api/openapi-ts`.

### Full Local CI

```bash
bash scripts/run-ci-local.sh         # mypy, ty, ruff, pytest, sphinx, bun lint, bun typecheck
```

## Architecture

### Backend (`acere/`)

- **`acere/main.py`** — FastAPI app with async lifespan. Creates two apps: main `app` (with compression) and `hls_app` (no compression, mounted at `/`). On startup, initializes DB, runs Alembic migrations, and creates all service singletons.
- **`acere/instances/`** — Global singletons. Each module has a private `_thing: T | None` with `get_*()` / `set_*()` functions. PLW0603 is intentionally disabled here. Uses `TYPE_CHECKING` guard pattern (`if TYPE_CHECKING: from X import Y; else: Y = object`) to break circular imports.
- **`acere/services/`** — Business logic: AcePool (stream pool management), AceScraper (scraping sources), EPGHandler (program guide), AceQuality (stream quality), RemoteSettingsFetcher, IPTV proxy, Xtream Codes support.
- **`acere/api/routes/`** — FastAPI routers split into `api/` (v1 REST API), `hls.py` (HLS streaming proxy), `iptv/` (m3u8/EPG endpoints), `frontend/` (serves built React app).
- **`acere/core/config/`** — pydantic-settings `BaseSettings`. Priority: init > dotenv > env vars (`ACERE_` prefix) > JSON config file. Testing mode (`ACERE_TESTING=1`) skips dotenv loading.
- **`acere/database/`** — SQLModel/SQLite with programmatic Alembic (no `alembic.ini`; config constructed in `migration/runner.py`). Models: user, acestream, category_xc, quality_cache.
- **`acere/cli/`** — CLI entry points: scraper, password-reset, db-migrate, debug tools.

### Frontend (`frontend/`)

- React 19 + TypeScript + Vite + Chakra UI v3
- TanStack Router (file-based routing with auto code splitting) + TanStack Query
- `frontend/src/client/` — Auto-generated from OpenAPI spec. **Do not edit manually.**
- `frontend/src/routeTree.gen.ts` — Auto-generated by TanStack Router plugin.
- Path alias: `@/*` → `./src/*`
- Biome for linting/formatting (double quotes, space indent, semicolons as-needed)

### Frontend Serving

When `FRONTEND_HOST=""`, FastAPI serves the built React frontend directly (files loaded into memory at startup, routes from `generated_frontend_paths.json`). When `FRONTEND_HOST` is set, it proxies to a separate dev server (Vite at port 5173).

## Testing

- `ACERE_TESTING=1` must be set BEFORE any acere imports — `conftest.py` handles this.
- Sockets are disabled by default (`--disable-socket`), only `127.0.0.1` and `127.0.1.1` are allowed.
- Test config: `tests/configs/test_valid.json`, test superuser password: `pytestpassword123`.
- Async mode is auto (pytest-asyncio).

## Code Style & Linting

- **Python**: Ruff with ALL rules selected (specific ignores listed in `pyproject.toml`). Line length 120. Double quotes, space indent, Google-style docstrings. Target: `py314`.
- **TypeScript**: Biome with recommended rules. Strict tsconfig (`noUnusedLocals`, `noUnusedParameters`, `noUncheckedIndexedAccess`).
- **Type checking**: Both mypy (strict) and ty are enforced. The `TYPE_CHECKING` guard pattern with `else: Y = object` is intentional — do not remove it.
- **Runtime typing**: `keep-runtime-typing = true` in ruff — do not move pydantic/SQLModel type annotations behind `TYPE_CHECKING` guards.
